# YouTube MP3 Downloader (ffmpeg 同梱版)

このアプリは、**YouTube の動画を MP3 に変換してダウンロード**する Windows 用デスクトップアプリです。  
**ffmpeg を含んでいるため、別途インストール不要**で、そのまま動作します。

---

## 📦 内容物

- `youtube-mp3.exe` … メイン実行ファイル
- `ffmpeg/bin/ffmpeg.exe` … 音声変換用の ffmpeg
- その他スクリプトやファイル（必要に応じて）

---

## 💻 動作環境

- Windows 10 / 11
- exe ファイル単体で動作可能
- インターネット接続必須（YouTube へのアクセス）

---

## 🚀 使い方

1. distフォルダ内にある`youtube-mp3.exe` をダブルクリックして起動
2. 表示された画面の **「YouTube URL」** 欄に動画 URL を入力（共有からURLをコピー）
3. **「Download MP3」** ボタンをクリック
4. 保存先フォルダを選択
5. ダウンロードと MP3 変換が開始されます
6. 完了すると、進捗バーが終了してメッセージが表示されます

---

## ⚙️ 注意事項

- ffmpeg は exe に同梱済みなので、別途インストール不要
- URL が正しくない場合や YouTube がアクセス制限されている場合はエラーになります
- ダウンロード中は画面を閉じないでください
- exe のファイルサイズは約 50〜60MB です

---

## 📌 トラブルシューティング

| 問題 | 対応 |
|------|------|
| アプリが起動しない | Windows Defender やアンチウイルスでブロックされていないか確認 |
| ダウンロードできない | インターネット接続と URL を確認 |
| ffmpeg not found エラー | exe 同梱版を使用している場合は発生しません |
