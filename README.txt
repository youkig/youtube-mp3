　　FFmpeg の導入方法（Windows版）
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

このアプリでは、音声変換に「FFmpeg」というソフトを使用します。
以下の手順で一度設定すれば、今後は自動的に動作します。

────────────────────────────
① FFmpeg をダウンロード
────────────────────────────
1. 下記サイトを開きます：
　https://www.gyan.dev/ffmpeg/builds/

2. ページ中ほどの「Release builds」欄から
　「ffmpeg-release-essentials.zip」をクリックしてダウンロードします。

────────────────────────────
② ZIPファイルを展開
────────────────────────────
1. ダウンロードした「ffmpeg-release-essentials.zip」を右クリック
2. 「すべて展開」を選択
3. 展開後、「ffmpeg-xxxx-win64」フォルダが作成されます
　（xxxxの部分はバージョン番号）

────────────────────────────
③ フォルダを移動
────────────────────────────
1. 展開したフォルダを任意の場所に移動します。
　例： C:\ffmpeg

────────────────────────────
④ 環境変数（PATH）の設定
────────────────────────────
1. Windowsの検索バーで「環境変数」と入力
2. 「システム環境変数の編集」を開く
3. 右下の「環境変数(N)...」ボタンをクリック
4. 「システム環境変数」欄の中から「Path」を選択 →「編集」
5. 「新規」をクリックし、次のパスを追加します：
　C:\ffmpeg\bin
6. 「OK」を押して閉じる

────────────────────────────
⑤ 動作確認
────────────────────────────
1. Windowsキーを押して「cmd」と入力 → コマンドプロンプトを開く
2. 次のコマンドを入力します：

　ffmpeg -version

3. バージョン情報が表示されたら成功です🎉

────────────────────────────
💡補足
────────────────────────────
・設定は一度行えば今後ずっと有効です。
・コマンドが「認識されません」と出る場合は、
　パソコンを再起動してください。
・アプリを起動する前に、必ずこの設定を完了しておいてください。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
　設定完了後、アプリの音声変換機能が正常に動作します。
